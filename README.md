# ServerPulse 🚀

**轻量级多云服务器资源监控与告警工具箱**  
一键批量采集 CPU/内存/磁盘使用率，多线程并行，支持钉钉、企业微信告警，pandas 数据分析，完美融入 CI/CD。


## ✨ 特性
- **多线程并发采集**：大幅缩短大批量服务器巡检时间
- **插件化告警**：钉钉群机器人、企业微信机器人，可自由扩展邮件、飞书等
- **灵活配置**：YAML 文件管理服务器、阈值、采集命令，支持密钥认证
- **智能分析**：自动计算全平台均值，标记超过阈值的异常项，并导出 CSV
- **完善的日志**：双输出（文件+控制台），DEBUG 级日志方便排错
- **生产可用**：异常隔离，单台故障不影响整体；上下文管理器保证 SSH 资源释放
- **CI 友好**：可通过 CLI 调用，集成到 GitHub Actions、Jenkins 等
